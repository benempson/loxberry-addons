**Why human:** Requires interactive UI testing.

**Test:** Save settings from the Settings tab, then navigate to the Logs tab.
**Expected:** A new log entry appears with `sev: Info`, `src: config`, `msg: Settings saved via web UI`.
**Why human:** Requires the PHP file path `LBPDATADIR` to resolve correctly in the running Loxberry environment.

INI format rules:
- Sections: [MQTT], [THRESHOLDS], [CRON], [NOTIFICATIONS], [EXCLUSIONS]
- Booleans: '0' or '1' (NOT 'true'/'false')
- Numerics: plain string digits (coerced by readConfig)
- EXCLUSIONS.devices: comma-separated IEEE addresses (no inline # comments -- ini@5.x treats # as comment)
- ini@5.x is the Node.js parser

Loxberry PHP SDK patterns (from research):
- require_once "loxberry_system.php"; require_once "loxberry_web.php";
- $L = LBSystem::readlanguage("language.ini");
- LBWeb::lbheader($title, $helpurl, $helptemplate);
- LBWeb::lbfooter();
- Path constants: LBPCONFIGDIR, LBPDATADIR, LBPBINDIR, LBHOMEDIR
- Config_Lite: require_once LBHOMEDIR . '/libs/phplib/Config/Lite.php';
- jQuery Mobile 1.4.x loaded automatically (tabs, flip switches, fieldcontain)
